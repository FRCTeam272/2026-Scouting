# Data

We assume that data is in the 2026 FRC708 format, if the data is not in that format the scripts will not work

# Running

## Setup

```bash
# Only needed for first run
pip install -r requriments.txt

# Generate the HTML file (and a bunch of data/*.jsons)
python read_scouting.py
```
